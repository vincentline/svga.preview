# SVGA 预览与转换工具

一个功能强大的在线动画预览和格式转换工具，支持 SVGA、Lottie 和 YYEVA-MP4 等多种动画格式。

## ✨ 核心功能

### 📁 SVGA 模块
- **在线预览**：拖入 SVGA 文件即可实时预览动画效果
- **播放控制**：支持播放/暂停、进度拖拽、帧数显示
- **素材替换**：可替换动画中的图片素材，并导出新的 SVGA 文件
- **格式转换**：
  - 导出为 GIF 动图
  - 转换为带 Alpha 通道的 MP4 视频（YYEVA 格式）

### 📺 YYEVA-MP4 模块
- **在线预览**：支持左右并排和上下并排的双通道MP4视频
- **透明通道**：正确解析和显示Alpha通道，支持半透明效果
- **播放控制**：支持播放/暂停、进度拖拽、帧数显示
- **音频播放**：支持带音频的MP4文件播放
- **模式切换**：在播放MP4过程中可随时切换到其他格式，自动清理资源

### 🎨 交互体验
- **画布操作**：鼠标滚轮缩放（20%-500%）、左键/中键拖拽画布
- **实际尺寸**：按动画原始尺寸渲染，支持一键恢复 1:1 显示
- **背景切换**：透明格子、白色、黑色三种背景可选
- **响应式布局**：自适应不同屏幕尺寸

## 🚀 技术特点

- **纯前端实现**：基于 Vue.js 2，无需后端服务器
- **本地处理**：所有文件处理均在浏览器中完成，保护隐私
- **高性能**：使用 Canvas 渲染，支持大尺寸动画流畅播放
- **跨平台**：支持 Windows、macOS、Linux 等主流操作系统

## 💡 使用场景

- **设计验收**：快速预览 SVGA 动画效果，检查设计细节
- **素材调整**：替换动画中的图片素材，快速生成变体
- **格式转换**：将 SVGA 转换为 GIF 或 MP4，适配不同平台需求
- **开发调试**：查看动画帧数、尺寸、内存占用等详细信息

## 📦 快速开始

### 在线使用
访问 [SVGA 预览工具](https://svga.duohuo.net) 直接使用，无需安装。


## 🎯 基本操作

1. **加载动画**：将 SVGA 文件拖入页面中央的播放区域
2. **播放控制**：点击播放/暂停按钮，或拖动进度条
3. **缩放画布**：使用鼠标滚轮放大/缩小，点击"1:1"按钮恢复原始尺寸
4. **拖动画布**：按住鼠标左键或中键拖动画布位置
5. **替换素材**：点击"素材图替换"按钮，选择要替换的图片，然后导出新文件
6. **转换格式**：点击"导出GIF"或"转MP4"按钮，根据提示完成转换
