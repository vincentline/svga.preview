### 侧边弹窗样式统一方案

#### 1. 方案目标

* 将所有侧边弹窗样式统一管理，提高可维护性

* 区分左侧和右侧弹窗，分别定义公共样式

* 确保所有颜色支持暗黑模式

* 新建独立CSS文件，不影响现有drawer.css

#### 2. 面板类型分析

| 面板类型  | 位置 | 类名                    | 功能     |
| ----- | -- | --------------------- | ------ |
| 左侧边弹窗 | 左侧 | `.chromakey-panel`    | 绿幕抠图   |
| 右侧边弹窗 | 右侧 | `.svga-panel`         | SVGA转换 |
| 右侧边弹窗 | 右侧 | `.material-panel`     | 素材管理   |
| 右侧边弹窗 | 右侧 | `.gif-panel`          | GIF导出  |
| 右侧边弹窗 | 右侧 | `.dual-channel-panel` | 双通道MP4 |
| 右侧边弹窗 | 右侧 | `.standard-mp4-panel` | 标准MP4  |
| 右侧边弹窗 | 右侧 | `.to-svga-panel`      | 转SVGA  |

#### 3. 样式结构设计

```css
/* 1. 基础样式 - 所有侧边弹窗共有属性 */
.side-panel {
  /* 位置、尺寸、动画、z-index等公共属性 */
}

/* 2. 方向样式 - 左侧/右侧特有属性 */
.side-panel--left {
  /* 左侧定位、left属性、动画方向 */
}

.side-panel--right {
  /* 右侧定位、right属性、动画方向 */
}

/* 3. 容器样式 - 所有弹窗容器共有属性 */
.side-panel-container {
  /* 背景、边框、阴影、圆角等 */
}

/* 4. 组件样式 - 各弹窗特有样式 */
.chromakey-panel {
  /* 绿幕抠图面板特有样式 */
}

.svga-panel {
  /* SVGA转换面板特有样式 */
}

/* 5. 主题支持 */
body.dark-mode .side-panel-container {
  /* 暗黑模式样式 */
}
```

#### 4. 具体实现步骤

1. **新建CSS文件**：

   * 在 `docs/assets/css/` 目录下新建 `panel.css` 文件

   * 在 `index.html` 中引入该文件

2. **定义基础样式**：

   * 提取所有侧边弹窗的公共属性

   * 定义 `.side-panel` 类：position: fixed, top: 0, height: 85%, display: flex, z-index: 1000

3. **定义方向样式**：

   * `.side-panel--left`：left: -400px, transition: left 0.3s cubic-bezier(0.4, 0, 0.2, 1)

   * `.side-panel--left.show`：left: 0

   * `.side-panel--right`：right: -400px, transition: right 0.3s cubic-bezier(0.4, 0, 0.2, 1)

   * `.side-panel--right.show`：right: 0

4. **定义容器样式**：

   * `.side-panel-container`：背景色、边框、圆角、内边距、阴影等

   * 确保所有颜色属性都有暗黑模式对应

5. **迁移现有样式**：

   * 将 `styles.css` 中的所有侧边弹窗样式迁移到 `panel.css`

   * 重构为新的样式结构

   * 保持各面板的独特宽度和内部布局

6. **更新组件模板**：

   * 更新各组件的模板，使用新的类名结构

   * 例如：`<div class="side-panel side-panel--left chromakey-panel" :class="{'show': visible}">`

7. **测试验证**：

   * 确保所有侧边弹窗正常显示和动画

   * 验证暗黑模式下的样式正确性

#### 5. 预期效果

* 所有侧边弹窗样式统一管理，提高可维护性

* 代码复用性提高，减少冗余

* 清晰的层级结构，便于扩展新弹窗

* 完整的暗黑模式支持

* 不影响现有more-drawer的功能
