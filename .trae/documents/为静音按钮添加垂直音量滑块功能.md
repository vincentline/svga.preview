# 为静音按钮添加垂直音量滑块功能

## 实现方案

### 1. HTML 结构修改
- 在静音按钮旁边添加音量滑块容器
- 为正常模式和沉浸模式的静音按钮都添加滑块
- 滑块默认隐藏，hover 时显示

### 2. CSS 样式添加
- 为音量滑块添加垂直布局样式
- 实现滑块轨道、滑块手柄的视觉效果
- 添加平滑的显示/隐藏过渡动画
- 确保在暗黑模式下也有正确的显示效果

### 3. JavaScript 逻辑实现
- 在 Vue 实例中添加音量相关状态变量
- 为 `GlobalAudioManager` 添加音量控制方法
- 为各个播放器适配器添加音量控制接口
- 实现滑块的鼠标/触摸拖动逻辑
- 处理音量变化时的音频实例更新

### 4. 功能集成
- 将音量滑块与现有静音按钮逻辑集成
- 确保音量状态在不同模式间正确切换
- 保持与现有代码风格和架构的一致性

## 具体修改点

### HTML 文件 (`docs/index.html`)
- 在 `mute-btn` 元素后添加音量滑块容器
- 在 `mini-mute-btn` 元素后添加迷你模式的音量滑块容器

### CSS 文件 (`docs/assets/css/styles.css`)
- 添加 `.volume-slider-container` 相关样式
- 添加 `.volume-slider-track` 和 `.volume-slider-handle` 样式
- 添加 hover 显示/隐藏的过渡效果

### JavaScript 文件
- **`docs/assets/js/core/app.js`**: 添加音量状态变量和方法
- **`docs/assets/js/controllers/player-controller.js`**: 为 `GlobalAudioManager` 添加音量控制方法，为各个适配器添加音量接口

## 技术要点

1. **垂直滑块实现**: 使用 CSS 变换实现垂直拖动
2. **事件处理**: 同时支持鼠标和触摸事件
3. **音频管理**: 确保音量控制对所有音频实例生效
4. **用户体验**: 平滑的过渡动画和直观的操作反馈
5. **兼容性**: 确保在不同浏览器和设备上都能正常工作

## 预期效果

- 当鼠标悬停在可点击的静音按钮上时，会显示垂直音量滑块
- 用户可以通过拖动滑块调整音量大小
- 滑块会根据当前音量位置显示
- 静音状态与音量大小保持同步
- 支持正常模式和沉浸模式的音量控制
