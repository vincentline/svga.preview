⌨️ 快捷操作

  • 空格 = 暂停/恢复播放
  • 滚轮 = 缩放画布
  • Ctrl + 滚轮 = 精确缩放
  • 中键拖动 = 平移画布
  • 1:1 按钮 = 恢复原始尺寸
  • 拖拽进度条 = 跳转到任意帧

📥 支持的文件格式

SVGA 动画（.svga）

  • ✅ 支持 SVGA 2.0 格式
  • ✅ 支持素材替换、导出 GIF、转双通道 MP4
  • ❌ 不支持 SVGA 1.0

双通道 MP4（.mp4）

  • ✅ 自动识别左右/上下双通道布局
  • ✅ 保留音轨、转 SVGA、导出 GIF
  • 💡 适用于游戏/直播动画素材

普通 MP4（.mp4）

  • ✅ 支持扣绿幕后转 SVGA / 双通道 MP4

Lottie 动画（.json）

  • ✅ 支持预览
  • ✅ 支持转 SVGA、转双通道 MP4、导出GIF

序列帧（多个 .png）（开发中）

  • ✅ 拖入多张图片自动识别序列
  • ✅ 需要文件名包含数字（如 frame_001.png）
  • ✅ 可设置播放帧率（1-120 fps）
  • ✅ 可导出为 SVGA / 双通道 MP4 / GIF

📤 导出功能

导出 GIF

  • 支持自定义尺寸、帧率、质量
  • 支持选择帧范围（避免文件过大）
  • 💡 技巧：大视频建议降低帧率到 15fps

SVGA → 双通道 MP4

  • 输出左彩色 + 右 Alpha 的双通道视频
  • 可调节压缩质量（默认 70%）
  • 保留原始帧率

双通道 MP4 → SVGA

  • 提取序列帧重建 SVGA 文件
  • 适合游戏客户端使用

素材替换 & 导出新 SVGA

  • 点击"素材图"按钮查看所有图片素材
  • 支持搜索、复制素材名
  • 替换后可直接导出新 SVGA

🎨 界面操作

背景切换

  • 6 种纯色背景 + 透明网格
  • 检查动画透明度必备
  • 支持暗黑/白天主题切换

文件信息

  • 自动显示尺寸、帧率、时长、文件大小
  • 实时显示当前帧数

清空画布

  • 点击"清空"重新开始
  • 或直接拖入新文件覆盖

💡 使用技巧

1️⃣ AI 生成视频 → 客户端动画

AI 生成绿幕视频 → 拖入本工具 → 扣绿幕
                 ↓
        导出 SVGA / 双通道 MP4
                 ↓
          直接用于游戏/App


2️⃣ 快速检查透明度

  • 切换背景色查看边缘是否干净
  • 透明网格模式最直观

3️⃣ 减小 GIF 文件体积

  • 降低输出尺寸（如 750 → 375）
  • 降低帧率（30fps → 15fps）
  • 只导出关键帧范围

4️⃣ 素材替换批量操作

  • 使用搜索框快速定位素材
  • 复制素材名粘贴到设计工具

⚙️ 系统要求

  • 现代浏览器（Chrome 90+ / Edge 90+ / Firefox 88+ / Safari 14+）
  • 建议内存 ≥ 4GB（处理大文件时）
  • GIF 导出使用 Web Worker，不卡页面
