- ### 📚 使用帮助
- 
 -#### 📺 预览播放
- 拖文件就能看，支持 `.svga` / `.json` (Lottie) / `.mp4` (YYEVA)
- Ctrl+滚轮画布自由缩放，1:1恢复原尺寸
- 鼠标中键拖拽
- 播放/暂停、进度条拖拽、帧数显示一应俱全
- 6种背景色 + 透明网格，检查透明度必备
- 
- #### 🎨 SVGA 高级玩法
- **素材替换**：换掉动画里的图片，实时预览效果
- **导出新 SVGA**：替换完直接导出，一气呵成
- **导出 GIF**：SVGA 秒变 GIF，分享更方便
- 暂不支持SVGA1.0
- 
- #### 🎁 特色工具
- **大R头像框自助**：专属功能入口，左上角导航直达
- 
- #### 💡 小提示
- **文件大小**：SVGA/Lottie 文件信息自动显示（尺寸、帧率、时长）
- **性能优化**：GIF 导出使用 Web Worker，不会卡页面
- **兼容性**：需要现代浏览器（Chrome/Edge/Firefox/Safari 最新版）
